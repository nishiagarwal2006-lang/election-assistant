# 🗳️ ElectionIQ — AI-Powered Election Process Assistant

> A smart, interactive AI assistant that helps citizens understand the election process, timelines, and steps clearly.

[![Node.js](https://img.shields.io/badge/Node.js-18+-green.svg)](https://nodejs.org)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

---

## 🎯 Chosen Vertical

**Civic & Democracy Education** — Helping first-time voters, students, and community members understand how elections work in a clear, step-by-step, nonpartisan way.

---

## ✨ Features

- **AI Chat Assistant** — Context-aware, nonpartisan answers powered by Claude AI
- **Interactive Voter Journey** — 6-step visual timeline from eligibility to results
- **Google News Integration** — Real-time election news via Google Custom Search
- **Multilingual Support** — Translate responses into 8 languages via Google Translate
- **Voter Context Adaptation** — Tailors responses for first-time voters, researchers, etc.
- **Glossary & FAQ** — Built-in election terminology and common questions
- **Secure by Default** — Helmet, rate limiting, input validation
